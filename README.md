# Full Stack Open Course Repository

This repository contains my personal solutions and exercises for the **Full Stack Open** course, officially titled **Fundamentals of Web Development** by the Department of Computer Science at the University of Helsinki.

## 🌍 Course Overview

This course is an introduction to **modern web development with JavaScript**, focusing on building **single-page applications (SPAs)** using **React** and supporting them with **RESTful** and **GraphQL** web services implemented with **Node.js**.

### 📚 Topics Covered

- React fundamentals and component-based architecture
- RESTful API development with Node.js and Express
- GraphQL APIs
- TypeScript in modern JavaScript projects
- React Native for mobile development
- Debugging techniques for web applications
- Containerization with Docker
- Configuration and environment management
- Continuous integration and deployment
- Using databases (MongoDB, PostgreSQL)

## 👨‍💻 Prerequisites

Participants are expected to have:

- Solid general programming skills
- Basic knowledge of JavaScript, HTML, CSS, and databases
- Familiarity with Git and version control workflows
- Independence and persistence in problem-solving

Each folder contains exercises or mini-projects for the corresponding course part.
